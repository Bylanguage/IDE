拜语言IDE(集成开发环境，即：拜语言开发工具)，当前版本为单机版不访问网络，拜语言源代码会自动转译为 java、c#、javascript ，转译结束后自动编译。

c#：调用 MSBuild.exe 编译成可执行的 dll 和 exe 等可执行文件，之后自动布署在 IIS Express 上。java：调用maven打成jar包，后自动布署在Tomcat上。

也就是说，拜语言源代码会、自动转译、自动编译、自动布署，结束后就可以运行起来。

至于运行java还是c#程序则由用户选择。

新版IDE搭载单机转译器，支持转译:java、c#、javascript、SQL(ms sql server，mysql，oracle)数据库。
